1.Сравнение ArrayList и LinkedList.
Исходя из произведенных тестов LinkedList имеет преимущество в скорости в операциях над элементами,
располагающимися в начале списка. Во всех остальных случаях преимущество у ArrayList.
2.Сравнение HashSet, LinkedHashSet и TreeSet.
Исходя из произведенных тестов TreeSet уступает остальным по всем операциям. TreeSet рекомендуется для 
хранения остортированных элементов, LinkedHashSet для хранения элементов в порядке добавления.
3.Сравнение HashMap, LinkedHashMap и TreeMap.
Аналогично среди Map по скорости во всех операциях уступает TreeMap. Однако позволяет хранить
элементы в остортированном порядке, равно как LinkedHashMap позволяет хранить элементы в порядке их добавления.
